## E2E User Flows

### Flow 1: First-Time Setup ✓ Complete
`claude --init` → SessionStart hook → config check → /install skill → just setup → setup.sh → vault configured → stop hook verifies

### Flow 2: Publish Posts ✓ Complete
`just publish` → read config → find posts → validate frontmatter → copy files → lint → commit → build → push

### Flow 3: List Posts ✓ Complete
`just list-posts [--all|--published]` → read config → find posts → validate → show table with status

### Flow 4: Unpublish Post ✓ Complete
`just unpublish <file>` → find in blog dir → confirm → git rm → commit (no push)

### Flow 5: Skills Oversight ✓ Complete
`/publish` → skill guides user → calls just publish → stop hook verifies build
